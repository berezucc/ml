# ml

- `paper-notes/` — reading notes on papers
- `implementations/` — small from-scratch implementations of ML algorithms
- `experiments/` — hypothesis-driven experiments

- `notes/` — topic notes, cheatsheets, and references

Polished, benchmarked projects are maintained in separate repositories.

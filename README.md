# ml

Machine learning notes, implementations, experiments, and coursework.

## Contents

- `paper-notes/` — reading notes on papers
- `implementations/` — small from-scratch implementations of ML algorithms
- `experiments/` — hypothesis-driven experiments
- `coursework/` — homework and projects from machine learning courses
- `notes/` — topic notes, cheatsheets, and references

Polished, benchmarked projects are maintained in separate repositories:

- [speculative-decoding](https://github.com/berezucc/speculative-decoding) — from-scratch speculative decoding on Apple M2 Max; 1.16× over greedy in PyTorch+MPS, 2.69× with MLX
- [hft-jump-diffusion](https://github.com/berezucc/hft-jump-diffusion) — optimal acquisition/liquidation under Semi-Markov and Hawkes jump-diffusion models; PDE solvers, Monte Carlo simulation, Streamlit dashboard
